Manitoba Weather Dashboard

Portfolio project demonstrating data analysis and visualization using Microsoft Power BI.

Interactive Power BI Report

This dashboard analyzes historical Manitoba weather data and includes interactive slicers, trend visualizations, and station-level filtering. Due to Power BI’s security and authentication requirements, the interactive version is hosted directly within the Power BI service.
